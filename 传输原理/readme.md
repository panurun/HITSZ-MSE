# 《传输原理》课程指南

## 课程基本信息
- **成绩构成**：采用**平时成绩**与**期末考试成绩**按 **5:5** 的比例核算总评成绩。
- **平时成绩组成**：包括**平时作业**、**课堂考勤**以及一项**思维导图大作业**。
- **课程形式**：包含常规授课与若干次**翻转课堂**环节。

## 考核方式说明
1.  **考试形式**：期末为**半开卷考试**，允许携带一张**A4纸**（内容手写或打印均可）进入考场。
2.  **考试难度**：试题较为**简单基础**，侧重概念与原理的理解，不涉及复杂计算。
3.  **考前指导**：最后一堂课会进行**作业讲解**并明确**考试注意事项**，是考前复习的关键。

## 学习与备考建议
- **核心准备**：充分利用**半开卷**规则，精心整理并携带好核心公式、概念和知识体系的A4纸。
- **平时积累**：认真完成作业，积极参与翻转课堂，并重视思维导图大作业，稳固平时成绩。
- **复习侧重**：紧跟最后一次课的指导，以PPT和作业题为基础，巩固基础概念与原理。
