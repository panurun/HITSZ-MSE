# 《材料分析方法》课程指南

## 课程基本信息
- **授课教师**：**陈祖煌**、**武俊伟**（两位老师共同授课）。
- **成绩构成**：总评成绩按 **平时成绩 20% + 期末考试成绩 80%** 计算。
- **平时成绩说明**：
    - **陈祖煌老师**部分：无课后作业，**平时分主要取决于课堂小测**。
    - **武俊伟老师**部分：有课后作业，**平时分主要取决于作业完成情况**。

## 课程与考试特点
1.  **课程内容**：本课程**内容量极大**，知识点繁多，需注意知识体系的构建与归纳。
2.  **考试重点**：期末考试**必考XRD（X射线衍射）及三强线相关内容**，是核心考点。
3.  **考前复习**：武俊伟老师会在**最后一节课**，就其讲授部分进行**复习串讲（提供复习PPT）**，此环节至关重要。
4.  **考核关联**：根据2025年情况，本课程与《物理化学》期末考试时间安排**相距较近**。

## 学习与备考建议
- **复习规划**：建议在复习《物理化学》的同时，同步安排本课程的复习，避免时间冲突。
- **重点把握**：必须**透彻掌握XRD及三强线**的全部原理、应用与分析方法。
- **平时积累**：针对不同老师的要求，分别重视**课堂小测（陈）** 和**课后作业（武）**，确保平时分。
- **资料整合**：充分利用武老师最后的复习课与PPT，系统回顾，梳理庞杂的知识点。
