# 《物理化学》课程指南

## 课程基本信息
- **成绩构成**：总评成绩按 **平时成绩 30% + 期末考试成绩 70%** 计算。

## 考试与课程特点
1.  **考试难度**：课程考试**内容相对基础，难度不大**。
2.  **核心题源**：**必须掌握课上提供的Word文档中的习题**，考试会直接出现**原题或高度相似的题目**。
3.  **授课重点**：老师在课堂上会**明确侧重讲解考试涵盖的内容**，需紧跟课堂节奏。
4.  **内容关联**：本课程与《材料化学》、《材料科学基础》等课程**内容重复度较高**，可联动复习。

## 学习与备考建议
- **首要任务**：**反复练习并彻底弄懂Word文档中的所有题目**，这是备考的基石。
- **课堂效率**：重点关注老师在课上强调的知识点与例题，这些很可能就是考试重点。
- **联动复习**：结合《材料化学》与《材料科学基础》的相关内容进行整合复习，可事半功倍。
