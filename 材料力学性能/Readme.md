# 《材料力学性能》课程指南

## 课程基本信息
- **课程定位**：本课程以《材料力学》为基础，更侧重于工程实际应用。
- **开课时间**：大三上学期（为同期五门考试课中课时最短的一门）。
- **成绩构成**：**最终成绩 = 期末考试成绩 × 70% + 平时成绩 × 30%**

## 考试与评分说明
1.  **考试范围**：试题内容完全来源于课程PPT，会涉及PPT中的具体细节。
2.  **难度与评分**：试卷整体难度不高，但存在明显的**压分现象**。学生普遍反映最终成绩比个人预估低5分左右，请对分数分布有合理预期。

## 学习与备考建议
- **核心资料**：课程PPT是复习备考的**唯一核心资料**。
- **复习重点**：在全面复习的基础上，务必关注PPT中的细节知识点。
- **心态管理**：鉴于压分情况普遍，应更关注知识掌握本身，并合理预估成绩。
