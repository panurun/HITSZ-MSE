# 《材料物理性能》课程指南

## 课程基本信息
- **成绩构成**：**最终成绩 = 平时成绩 × 30% + 期末考试成绩 × 70%**
- **核心内容**：课程主要涵盖**热学**、**磁学**、**光学**等四个部分的物理性能。

## 考试与评分说明
1.  **考试范围**：考题全部来源于**课程PPT**与**课后作业**。
2.  **复习资源**：最后一节课会提供**复习PPT**并进行**作业讲解**，是备考的关键。
3.  **考试难度**：整体难度较低。根据2025年考试情况，试卷中**未设置计算题**。
4.  **给分情况**：本课程考试**给分普遍偏高**，整体成绩较为理想。

## 学习与备考建议
- **核心资料**：紧跟课堂PPT，并**透彻完成所有作业**。
- **复习要点**：最后一节的复习课内容至关重要，务必重点掌握。
- **备考策略**：可围绕PPT与作业题进行系统性复习，鉴于无计算题，应侧重对概念、原理和结论的理解与记忆。
